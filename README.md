# FinScope

Basit, Vercel'e deploy edilebilir canlı piyasa dashboard'u.

## Dosyaları GitHub'a yükleme

1. Bu ZIP'i bilgisayarına indir.
2. ZIP'i aç.
3. GitHub'da Finscope reposuna gir.
4. `uploading an existing file` bağlantısına tıkla.
5. ZIP'in içinden çıkan klasörün içindeki dosya ve klasörleri sürükle:
   - `app`
   - `package.json`
   - `next.config.js`
   - `tsconfig.json`
   - `.gitignore`
   - `README.md`
6. `Commit changes` butonuna bas.
7. Vercel'e dön, projeyi tekrar deploy et.

## Not

Bu demo döviz için gerçek açık API kullanır. Altın/gümüş ve BIST kaynakları yanıt vermezse sayfa çalışmaya devam etmek için yedek değer gösterir.
